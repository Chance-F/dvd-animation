## Run

Clone repository:

```sh
git clone https://github.com/remarkablemark/Bouncing-DVD-Logo.git
cd Bouncing-DVD-Logo
```

Start a static server:

```sh
python -m SimpleHTTPServer
```

Open http://localhost:8000/:

```sh
open http://localhost:8000/
```

To stop the server, press `Ctrl + C`.
